# Web Scraping Task

This project demonstrates web scraping using Python, BeautifulSoup, and Pandas.

## Objective
Extract book data from a website and store it in a structured CSV format.

## Tools Used
- Python
- BeautifulSoup
- Pandas
- Requests

## Output
A CSV dataset containing:
- Book Title
- Price
- Availability
